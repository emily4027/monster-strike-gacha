# 📊 怪物彈珠 - 抽蛋統計工具 (Monster Strike Gacha Tracker)

### 這是一個專為《怪物彈珠》多帳號玩家設計的統計工具，用於記錄、管理並分析各個帳號在不同活動中的

### 抽蛋投入狀況。具備詳細的歷史紀錄、數據儀表板與備份功能。

## ✨ 主要功能

## 1. 📝 紀錄管理系統

### 雙模式介面：

### 檢視模式：純瀏覽數據，防止誤觸修改。

### 編輯模式：解鎖表單與編輯/刪除按鈕。

### 彈性錄入：

### 動態新增：遇到新活動或新帳號時，可直接在選單中選擇「-- 新增 --」進行建立，無需切換頁

### 面。

### 活動標籤：支援設定活動類型（本家、合作、免費、寶珠），並在列表中以不同色條標示。

###  智能編輯：

### 單筆編輯：修正特定帳號的抽數或備註。

###  批次更新 (Batch Update)：若活動名稱或日期有誤，可一次更新所有帳號在該活動的紀錄，省去逐筆修改的麻煩。

## 2. 📈 統計儀表板
###     即時數據分析：
###        總投入統計：顯示符合當前篩選條件的總抽數。
###        帳號排行：自動計算各帳號的累積抽數並排序。
###        活動歷程：依時間軸列出各活動的總投入量，支援滾動瀏覽。
###     多維度篩選：可交叉篩選 特定帳號 + 特定年份 + 特定月份，快速回顧特定時期的抽卡狀況。

## 3. 💾 資料安全與遷移
###      本地儲存：使用 LocalStorage 自動保存，無須聯網與登入。
###      完整備份：
###         匯出：將所有紀錄打包為 JSON 檔案。
###         匯入：支援讀取備份檔，方便在不同裝置間轉移資料。
###      資料清除：提供一鍵重置功能（具備防呆確認）。

## 4. 🎨 介面設計
###      現代化 UI：基於 Bootstrap 5 進行客製化，採用卡片式設計與毛玻璃效果。
###      視覺化回饋：
###         表格左側根據標籤（本家/合作/免費）顯示對應色條。


```
操作時皆有 Toast 訊息提示（成功/錯誤/警告）。
RWD 響應式：在手機與電腦上皆能完美顯示。
```

## 🚀 如何使用

### 1. 初始化

```
開啟 index.html。
首次使用會載入範例資料（若有設定）。
```
###  2. 新增紀錄
    點擊右下方的「檢視模式 (點此編輯)」切換為編輯模式。
    填寫日期、選擇/輸入活動、選擇/輸入帳號、抽數與獲得角色。
    點擊「儲存紀錄」。
###  3. 編輯/修正
    在下方列表找到該筆紀錄，點擊黃色「鉛筆」按鈕。
    系統會將資料帶入上方表單，修改後點擊「更新紀錄」。
    提示：若選擇「 _- -_ 全部帳號 _(_ 僅編輯 _) - -_ 」，可同時修改該活動所有帳號的日期或名稱。
###  4. 查看統計
    使用中間的儀表板區塊。
    切換「帳號」、「年份」下拉選單，下方的總數與列表會即時連動更新。

```

├── style.css # 客製化樣式與動畫效果

```
## 📝 注意事項

### 瀏覽器快取：資料儲存於瀏覽器中，清除快取可能會導致資料遺失。請務必定期使用「匯出備份」功

### 能。


### 活動名稱格式：系統會嘗試自動簡化顯示名稱（例如隱藏年份），建議輸入完整名稱（如：2024-05月

### 鬼滅之刃合作）以利排序。
